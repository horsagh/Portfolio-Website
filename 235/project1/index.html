Ruins

2/2019-4/2021

Ruins is a CS:GO level created in the hammer editor. It was a solo personal project. I designed and iterated on the layout for engaging and dynamic gameplay. I conducted multiple playtests and analyzed popular maps extensively. I detailed the map with assets and textures from other source levels.

Clone!

4/2022-Present

Clone! is a web puzzle/stealth game written in JavaScript using PIXIJS. It was a solo academic project, that I've decided to keep working on. The game's core mechanic is cloning- you can use your clones as bait to move enemies into vulnerable positions. I developed and utilized multiple collision detection algorithims for this game, including AABB, circle-circle and a line trace function.

Rune Smith

1/2022-5/2022

Rune Smith was a turn-based Roguelite deckbuilding game that I developed in Unity using C# alongside 6 other students. I primarily worked on our rune sequencing system and our UI art.

The Trials of Persephone

3/2022-Present

The Trials of Persephone is a VR adventure game developed in Unreal Engine 4. Its a personal project that I am developing with another student. The game rewards interaction with the environment by allowing the player to collect mana from trees and rocks around them, that they can then use to cast spells. I did all of the scripting for this project thusfar, as well as the rigging, animations, and vfx.

The Halls

1/2021-5/2021

The Halls is an endless topdown roguelike written in C# using MonoGame. It was an academic project that I worked on with 3 other students. I created the room sequencing system, AI, and player mechanics. I also implemented the UI and animations.
